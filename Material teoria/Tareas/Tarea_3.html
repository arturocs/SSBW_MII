Frameworks CSS

Para la interface, usaremos algún CSS Framework que nos facilite la tarea. Además así, conseguiremos un diseño 'responsive' , que se adpate automáticamente al tamaño de la pantalla.

El framework más popular es Bootstrap, que es el que se menciona en el resto de este guión.

Para empezar a utilizarlo:

  1. -   Descargamos el código desde Bootstrap y lo descomprimimos, o usamos npm,
    o simplemente incluimos el enlace de un Content Delivery Network

  2. -   Copiamos y pegamos el código mínimo desde Basic template o desde ejemplos

Para no repetir código html en los distintos templates que tengamos, hay que aprovechar la herencia de las plantillas de django
