REST API

En esta tarea añadiremos una API RESTFULL a nuestra aplicación, para poder conectar con ella con una SPA o desde una aplicación móvil.

Para ello utilizaremos la extensión Django REST Framework, que facilita la tarea con una clase parecida a forms, serializers, que transforma los datos de objetos a/desde JSON. Tambien tiene ModelViewSets, que conecta los serializadores con el model, y routers que diregen las peticiones a los viewsets. Podemos orientarnos por el tutorial para hacer la apliación

Los endpoints que habilitaremos serian:

  • /api/excursiones         GET lista todas las excursiones
  • /api/excursiones         POST añade una excursión
  • /api/excursion/<id>   GET detalle de una excursión
  • /api/excursion/<id>   PUT modifica una excursión
  • /api/excursion/<id>   DELETE borra una excursión

Hay también que poner restricciones de acceso a la BD según los usuarios. En How to Use JWT Authentication with Django REST Framework, cuentan como hacerlo
