CRUD

Para esta tarea añadiremos una vista de detalle para cada excursión, así como la posibilidad de añadir, borrar, y editar registros.

Antes completaremos la BD para poder incluir una imagen.

Para los formularios usar la clase prevista en django, para poder subir los archivos, el formulario sería: 

<form action="{% url 'añadir' %}" method="post" enctype="multipart/form-data">
	{% csrf_token %}
	<table>
		{{ form.as_table }}
	</table>
	<button type="submit" class="btn btn-warning">Añadir</button>
</form>

Los datos de vuelta del formulario están en request.POST y request.FILES,  ( File Upload en la documentación de Django )
